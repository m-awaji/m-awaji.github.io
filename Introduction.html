Description logics (DLs)

Description logics (DLs) are a family of logic-based knowledge representation formalisms that are used in artificial intelligence to describe relevant concepts of a domain of interest in a well-structured method [1].

DLs have been used in different application domains such as: configuration of telecommunications, software information and documentation systems and databases .

There are many varieties of Description Logics including: AL (Attributive language), FL (Frame based description Language), EL (Existential language), etc. Each variant has specific constructors and its expressive power is constrained by the constructors it can use.

ALC

In this study you will learn notations, syntax and semantics of the basic Description Logic called ALC (Attribute Language with Complement).

ALC is at the core of the Semantic Web Ontology Languages (OWL).
