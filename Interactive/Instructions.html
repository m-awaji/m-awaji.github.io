General instructions:

Please read instructions carefully before starting the exercise.

• In this study, we want to administer anonymous tests where you answer two exercises before and after the study (we call them pre-test and post-test). At the same time, we want to compare your score in the pre-test with your score in the post-test; without linking them with your confidential data such as name and email.

• To achieve this, you are asked to generate a random ID, and use it in both tests so that we can match pre-test responses with post-test responses.

• No one will know what is your ID, so please save it in an accessible place to you to use it later.

Now, click the button below to generate your ID.


   

Click save to save the ID to your local computer.

Tests instructions:

  1. You should spend arounf 30 minutes or less to complete each test.
  2. Each test contains four multiple choice questions, each with four choices.
  3. Some questions have multiple answeres, select all answers that apply.
  4. Answer all questions, then click submit.
