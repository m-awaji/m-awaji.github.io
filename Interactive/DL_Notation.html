DL Notation

Like any formalism or computer language, Description Logics have specific syntax and semantics.

But, before talking about the syntax and semantics of Description Logic, let's assume that we want to describe a university domain which is populated with a number of elements such as:

Cs1, Paul, Professor, Sara, Tech1, teaches, Pro1, Research, publishes, Course, etc.

In Description Logics, all elements in the domain are classified as being one of two main building blocks.

In the following sections, we will: introduce the main building blocks and where to use them. Then, we will describe the syntax of ALC. Finally, we will focus on the semantics and graphical representations of interpretations.
