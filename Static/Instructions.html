General instructions:

Please read instructions carefully before starting the tests.

• In this study, we want to administer anonymous tests where you answer one test before accessing the learning materials and another test after accessing the learning materials (we call the tests pre-test and post-test, respectively).

•At the same time, we want to compare your score in the pre-test with your score in the post-test; without linking them with your confidential data such as name and email.

• To achieve this, you are asked to generate a random ID, and use it in both tests so that we can match pre-test responses with post-test responses.

• No one will know what is your ID, so please save it in an accessible place to you to use it later.
