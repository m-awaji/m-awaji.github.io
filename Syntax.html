How to build concept descriptions (compound concepts)?

We already know what are the concept names and the role names and you may have been able to identify them in Examples 3 and 4.

Now, to build concept descriptions we need to learn the syntax, i.e. how to form a compound concept in ALC and the semantics, i.e. the meaning.

ALC Syntax

Syntax: is the set of rules that defines the combinations of symbols that are legal in ALC.

For example, if we want to describe a Male Professor, we need to use constructors to build a compound concept that is a conjunction of both Male concept and Professor concept.

The following table shows the constructors supported by ALC for C and D as concept names and R as a role.

Constructor Syntax Example Meaning
conjunction C ⊓ D Professor ⊓ Male The set of elements that are both Professors and Males.
disjunction C ⊔ D Professor ⊔ Doctor The set of elements that are either Professors or Researchers.
negation ¬C ¬Student The set of all elements that are not Students.
exist restrict. (Some) ∃R.C ∃teaches.Course The set of elements that teach(es) at least one Course.
value restrict. (All) ∀R.C ∀teaches.Cs1 The set of elements who teach(es) only Cs1.

Note 3:
A set of available constructors depends on the particular DL variant. In this study we are focusing only on the ALC variant of DL.
